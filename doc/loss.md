## 1. 物理背景：螺旋槽止推轴承

```
        R_lim[1] = 1.0 (外径 r_o = 52mm)
        ┌──────────────────────────┐
        │   螺旋槽 (Spiral Groove)  │  ← 对数螺线
        │   H = 4×h_i (深槽区)     │     θ_sym(R) = ln(R/r_g)/tan(α) + θ_offset
        │                          │
        │   台地区 (Plateau)        │
        │   H = h_i (浅槽区)       │
        │                          │
        └──────────────────────────┘
        R_lim[0] = 0.9038 (内径 r_i = 47mm)

        周向周期: K = 6 (360° 均分为 6 个周期)
        单周期角度域: θ ∈ [0, 2π/6] = [0, 1.047]
```

- **全膜区（Full Film）**：液体完全填充间隙，压力 p > 0，空化分数 γ = 0
- **空化区（Cavitation）**：液体破裂形成气泡，p = 0（= 空化压力），γ > 0 表示空隙率
- **JFO 条件**：p 和 γ 不能同时 > 0（同一位置不能同时是全膜和空化）

## 2. 无量纲化与关键参数计算

代码位置：[`reynold_pinn_torch.py:91-124`](../reynold_pinn_torch.py#L91-L124)

### 基准量

| 符号 | 公式 | 物理含义 | 数值 |
|------|------|----------|------|
| `r_base` | `= r_o` | 特征长度 | 0.052 m |
| `p_o` | `= 1.5 × p_i` | 外径压力 | 0.15 MPa |
| `p_base` | `= 10 × p_o` | **特征压力（关键！）** | **1.5 MPa** |
| `ω` | `= 6000 × 2π / 60` | 角速度 | 628.3 rad/s |

### 无量纲参数

| 符号 | 公式 | 数值 | 说明 |
|------|------|------|------|
| **Λ** | `6·η·ω·r_base² / (h_i²·p_base)` | **604.08** | **轴承数 — 核心参数** |
| P_i | `p_i / p_base` | 0.0667 | 无量纲内径压力 |
| P_o | `p_o / p_base` | 0.1000 | 无量纲外径压力 |
| R_lim | `[r_i/r_base, 1.0]` | [0.9038, 1.0] | 无量纲径向范围 |
| θ_lim | `[0, 2π/K]` | [0, 1.0472] | 单周期角度范围 |

### Λ 的计算展开

```
η = 8.00×10⁻⁴  Pa·s        (动力粘度)
ω = 628.32     rad/s       (角速度 @6000rpm)
r_base = 0.052  m           (外径)
h_i = 3.0×10⁻⁶  m           (平衡膜厚)
p_base = 1.5×10⁶ Pa         (特征压力)

Λ = 6 × 8e-4 × 628.32 × 0.052² / ((3e-6)² × 1.5e6)
  = 6 × 0.50266 × 0.002704 / (9e-12 × 1.5e6)
  = 0.008156 / 1.35e-5
  = 604.08
```

**Λ 的物理意义**：对流项（楔形效应）与扩散项（Poiseuille 流动）之比。Λ = 604 意味着**对流主导**——楔形效应远强于压力驱动的 Poiseuille 流动。这类似于流体力学中高雷诺数或高 Péclet 数的对流-扩散问题。

## 3. 膜厚函数 H(R,θ)

代码位置：[`reynold_pinn_torch.py:130-180`](../reynold_pinn_torch.py#L130-L180)

### 螺旋槽几何

```
槽线方程（对数螺线）：θ_sym(R) = ln(R / r_g) / tan(α) + θ_offset

参数：
  α = 3°               (螺旋角，tan α ≈ 0.0524)
  r_g = R_d_1 = 1.043 × r_i / r_base = 0.943
  groove_ratio = 0.5   (槽宽比)
  h_texture = 3 × h_i  (槽深 = 3 倍平衡膜厚)
```

### Sigmoid 平滑

使用 sigmoid 函数平滑沟槽边界过渡（避免导数不连续）：

```
径向平滑：  ξ_R  = (R_max - R_min) / R_ξ = 0.0962 / 100 ≈ 0.001   ← 非常陡
周向平滑：  ξ_θ  = (θ_max - θ_min) / N_ξ = 1.047 / 50 ≈ 0.021    ← 较陡

is_texture = sigmoid((R-R_d_1)/ξ_R) · sigmoid((R_d_2-R)/ξ_R)   ← 径向包络
           · Σ_k sigmoid((θ-θ_sym(R)+offset_k)/ξ_θ)            ← 周期性周向位置
             · sigmoid((θ_sym(R)-θ+2π/K·ratio-offset_k)/ξ_θ)   ← 槽宽

H(R,θ) = 1 × (1 - is_texture) + 4 × is_texture
```

### H 的关键性质

| 性质 | 数值 | 影响 |
|------|------|------|
| 台地区 H | 1 | H³ = 1 |
| 槽区 H | 4 | **H³ = 64** |
| H³ 对比度 | 64× | Poiseuille 项系数跨越两个数量级 |
| ∂H/∂θ (槽边界) | ~3 / 0.021 ≈ **143** | 楔形效应的源 |

## 4. 损失项一：雷诺方程残差 L_Reynolds

代码位置：[`reynold_pinn_torch.py:197-242`](../reynold_pinn_torch.py#L197-L242)

损失计算：[`models.py:202-214`](../torch_pinn/models.py#L202-L214) — `MSE(f_p, 0)`

### 4.1 极坐标雷诺方程（含 JFO 空化）

无量纲化后的雷诺方程在极坐标下表示为：

```
f_p ≡ part_1 + part_2 + part_3_1 + part_3_2 + part_stab = 0
```

其中各项的物理含义和计算方式：

---

#### 项 1：径向 Poiseuille 流动（扩散项）

```
term_1_inner = R · H³ · ∂p/∂R        (径向质量流量)
part_1 = [∂(term_1_inner)/∂R] / R    (径向净流出)

展开：part_1 = H³·∂²p/∂R² + (3H²·∂H/∂R + H³/R)·∂p/∂R
```

**计算图**：网络 → p → autograd(∂p/∂R) → H³·R·∂p/∂R → autograd(∂/∂R) → /R

这是**二阶导数**项。网格需要学习 p 的二阶导数来产生正确的扩散通量。

---

#### 项 2：周向 Poiseuille 流动（扩散项）

```
term_2_inner = H³ · ∂p/∂θ
part_2 = [∂(term_2_inner)/∂θ] / R²

展开：part_2 = H³/R² · ∂²p/∂θ² + 3H²/R² · ∂H/∂θ · ∂p/∂θ
```

同样是**二阶导数**项。注意 H³ 和 ∂H/∂θ 在这里出现——槽边界的 H 梯度会放大 ∂p/∂θ 的影响。

---

#### 项 3_1：楔形效应（对流源项）⚡

```
part_3_1 = −Λ · ∂H/∂θ
```

**这是整个方程的主导项，也是 L_Reynolds 巨大的根源。**

- `Λ = 604`：放大了 ~600 倍
- `∂H/∂θ` 在槽边界处 ≈ 3 / ξ_θ ≈ 143
- **`part_3_1 ≈ 604 × 143 ≈ 86,400`**

这个项**完全由几何决定**（不依赖于网络输出），它是固定的、不可消除的"驱动力"。网络必须通过调整 p 和 γ 使得其他四项之和**精确抵消**这个 ~86,400 的项。

---

#### 项 3_2：空化对流项

```
neg_gamma_H_theta = ∂(−γ·H)/∂θ
part_3_2 = −Λ · neg_gamma_H_theta = Λ · ∂(γ·H)/∂θ
```

- 这是网络**可以控制**的对流项（通过 γ 输出）
- 量级与 Λ × ∂(γ·H)/∂θ 相当，最多也是 O(10⁵)

---

#### 项 5：JFO 稳定化项

```python
div_gamma   = ∂γ/∂θ                      # 空化梯度
div_2_gamma = ∂²γ/∂θ²                    # 空化二阶导数
div_p       = ∂p/∂θ                      # 压力周向梯度

ε = 0.1
τ   = (|div_gamma| − div_gamma) × ε      # γ 下降时 > 0，上升时 = 0 (detach)
τ_2 = (div_p − |div_p|) × ε              # p 上升时 > 0，下降时 = 0 (detach)

part_stab = div_2_gamma · τ · τ_2
```

**物理含义**：在空化边界处（p 上升 + γ 下降，即 cavitation → full film 的再形成边界），加入人工扩散 `∂²γ/∂θ² · ε² · (∂γ/∂θ) · (∂p/∂θ)`。这类似于 CFD 中的迎风格式，防止空化边界处的数值振荡。

`τ` 和 `τ_2` 被 **detach**（与 TF 的 `stop_gradient` 对应），不参与梯度反传，仅作为系数。

---

### 4.2 完整雷诺方程

```
  H³·∂²p/∂R² + (3H²·∂H/∂R)(∂p/∂R) + (H³/R)(∂p/∂R)
+ H³/R² · ∂²p/∂θ² + (3H²/R² · ∂H/∂θ)(∂p/∂θ)
− Λ·∂H/∂θ                                                   ← O(10⁴~10⁵) 源项
+ Λ·∂(γ·H)/∂θ                                               ← O(10⁴~10⁵) 空化对流
+ ∂²γ/∂θ² · τ · τ_2                                         ← 稳定化项 (detach 系数)
= 0
```

### 4.3 为什么这个损失很难优化

| 挑战 | 具体数值 | 后果 |
|------|----------|------|
| **Λ 放大源项** | Λ=604, ∂H/∂θ≈143 → 源≈86,400 | MSE 初始可达 10⁸~10¹⁰ |
| **H³ 系数跨度** | 1 ↔ 64（64× 变化） | 不同区域的残差量级差异巨大 |
| **二阶 autograd** | 需要 ∂²p/∂R², ∂²p/∂θ² | 对网络光滑性要求极高 |
| **高对流主导** | Λ=604 ≫ 1 | 接近双曲型 PDE，解可能不光滑 |
| **自由空化边界** | γ 的分布是未知的 | 空化区的形状和位置需要同时学习 |

## 5. 损失项二：JFO 互补条件 L_FB

代码位置：[`reynold_pinn_torch.py:244-248`](../reynold_pinn_torch.py#L244-L248)

损失计算：[`models.py:248-264`](../torch_pinn/models.py#L248-L264) — `MSE(f_FB, 0)`

### 5.1 Fischer-Burmeister 互补函数

```
f_FB(p, γ) = p + γ − √(p² + γ²)

L_FB = MSE(p + γ − √(p² + γ²), 0)
```

### 5.2 数学性质

FB 函数是标准互补条件 `min(p, γ) = 0` 的半光滑等价形式：

| 条件 | 分析 | f_FB 值 |
|------|------|---------|
| p > 0, γ = 0 | `p + 0 − √(p²) = p − |p|` | **= 0** ✓ |
| p = 0, γ > 0 | `0 + γ − √(γ²) = γ − |γ|` | **= 0** ✓ |
| p > 0, γ > 0 | `p + γ − √(p²+γ²)` 由三角形不等式 `√(p²+γ²) < p + γ` | **> 0** ✗ |
| p = 0, γ = 0 | `0 − 0` | **= 0** ✓ |

**关键性质**：FB 函数是 Lipschitz 连续的（梯度有界），不像 `min(p, γ)` 那样不可导。这使得它可以安全地用于基于梯度的优化。

### 5.3 为什么 L_FB 天然很小

网络输出经过 [networks.py:416-422](../torch_pinn/networks.py#L416-L422) 的处理：

```python
predictions  = tanh(g_func + sigma_func_1 × p_raw)²     # 压力 P
prediction_g = tanh(sigma_func_2 × g_raw)²              # 空化 γ
```

**`tanh²` 激活天然保证了 p ≥ 0 和 γ ≥ 0**。

FB 条件唯一的约束是 "p 和 γ 不能同时 > 0"。这个约束相对容易满足：

1. **边界条件的引导**：内外径处 P_i=0.0667, P_o=0.1 > 0，BC 强制执行 p > 0
2. **物理的自然分离**：高压区（台地）→ p > 0；槽区（发散间隙）→ 压力下降 → 可能空化 → γ > 0
3. **FB 函数的宽容性**：当 p ≫ γ 或 γ ≫ p 时，f_FB 非常接近 0

**L_FB ≈ 10⁻³ 意味着 FB 残差的均方根 ≈ 0.03**。考虑到 p 和 γ 的范围是 [0, 1]，这个误差已经足够小，说明 JFO 条件基本满足。

## 6. 损失项三：边界条件 L_BC

代码位置：[`models.py:216-246`](../torch_pinn/models.py#L216-L246)

### 6.1 Dirichlet BC

在内径 R = R_lim[0] 和外径 R = R_lim[1] 处，强制压力：

```
L_BC = mean(MSE(p_pred_inner, P_i) + MSE(p_pred_outer, P_o))
```

### 6.2 BC 的硬执行机制

网络通过 [`Out_Imp_BC_layer`](../torch_pinn/networks.py#L106-L133) 和 [`Out_Imp_BC_value_layer`](../torch_pinn/networks.py#L139-L177) 实现了**硬边界条件**：

**sigma 函数（边界距离函数）**：
```
sigma(R_norm) = p₃ × (1 − exp(p₁×(−1−R_norm))) × (1 − exp(p₂×(R_norm−1)))
```
- 在 R = R_min (归一化 = −1)：sigma = 0
- 在 R = R_max (归一化 = +1)：sigma = 0
- 在内部：sigma > 0（可学习的形状）

**g_func（Hermite 插值）**：
- 在 R = R_min：g_func = P_i（内径 BC 值）
- 在 R = R_max：g_func = P_o（外径 BC 值）
- 在内部：平滑插值

**输出构造**：
```
P_raw = NN_output_pressure
P = tanh(g_func + sigma × P_raw)²
```
- 在边界处 sigma=0, g_func=BC 值 → P = tanh(BC值)² ✓
- 在内部 sigma>0 → 网络可以自由调整 P

这意味着**BC 是通过构造精确满足的**，不是通过惩罚项近似满足的。因此 L_BC 即使在训练初期也很小。

## 7. 为何 L_Reynolds ≈ 10⁵ 而 L_FB ≈ 10⁻³？

这是本项目**最核心的问题**。以下从多个维度分析这 10⁸ 倍（按 MSE 算）的差距。

### 7.1 量级来源的对比表

| 维度 | L_Reynolds | L_FB |
|------|-----------|------|
| **公式输入** | 网络二阶导数 + 膜厚函数 | 网络输出值 |
| **主导项量级** | Λ·∂H/∂θ ≈ **86,400** | p, γ ∈ **[0, 1]** |
| **MSE 初始** | ~86,400² ≈ **7.5×10⁹** | ~0.2² ≈ **4×10⁻²** |
| **MSE 收敛** | **3×10⁵ ~ 3×10⁶** | **~1×10⁻³** |
| **优化的"天花板"** | 受限于物理精度 | 受限于数值精度 |

### 7.2 详细分析：L_Reynolds 的量级

```
L_Reynolds = MSE(f_p, 0)，其中 f_p = part_1 + part_2 + part_3_1 + part_3_2 + stab

训练初期（网络未学习）：
  ┌─────────────────────────────────────────────────────────────┐
  │ part_1 (径向扩散) ≈ 小/随机          ~ O(1~100)             │
  │ part_2 (周向扩散) ≈ 小/随机          ~ O(1~100)             │
  │ part_3_1 (Λ·∂H/∂θ) ≈ 86,400          ← 主导！              │
  │ part_3_2 (Λ·∂(γH)/∂θ) ≈ 小/随机      ~ O(1~100)            │
  │ stab (稳定项) ≈ 小                    ~ O(1)                │
  ├─────────────────────────────────────────────────────────────┤
  │ f_p ≈ 86,400 + (小量)                 ~ O(10⁴~10⁵)          │
  │ MSE(f_p, 0) ≈ 86,400²                ~ 7.5×10⁹             │
  └─────────────────────────────────────────────────────────────┘

训练后期（网络部分学习）：
  ┌─────────────────────────────────────────────────────────────┐
  │ f_p ≈ 86,400 − (网络产生的抵消项)     ~ O(500~1000)         │
  │ MSE(f_p, 0)                           ~ 2.5×10⁵ ~ 1×10⁶    │
  └─────────────────────────────────────────────────────────────┘
```

**网络需要产生 ~86,400 的二阶导数来抵消源项**。但由于：
- p 被 tanh² 限制在 [0, 0.1] 左右（对应于 BC 值 0.0667~0.1）
- 域的大小是 R∈[0.9038, 1.0]（ΔR=0.096）和 θ∈[0, 1.047]（Δθ=1.047）
- 二阶导数 `∂²p/∂R²` 的数量级 ≈ Δp/(ΔR)² ≈ 0.03/0.009 ≈ 3
- H³ 可以放大到 64×，所以 H³·∂²p/∂R² ≈ 64×3 ≈ 192

而我们需要 ~86,400。差距是 **86,400 / 192 ≈ 450 倍**。

这意味着：**或者 p 的二阶导数需要远超物理范围，或者网络需要学会非常精细的压力场结构来产生大梯度**。实际情况中，沟槽边界的压力梯度确实可以很大（由于 H³ 的突变），但即便如此，精确抵消 Λ·∂H/∂θ 仍然极难。

### 7.3 详细分析：L_FB 的量级

```
L_FB = MSE(p + γ − √(p² + γ²), 0)

训练全程：
  ┌─────────────────────────────────────────────────────────────┐
  │ tanh² 保证 p ∈ [0,1], γ ∈ [0,1]                             │
  │ 在 99%+ 的区域，p 和 γ 天然分离（一边主导）                  │
  │ f_FB 仅在 p ≈ γ > 0 的区域显著 ≠ 0                          │
  │                                                             │
  │ 最坏情况 p = γ = 0.5:                                       │
  │   f_FB = 0.5 + 0.5 − √(0.25+0.25) = 1 − 0.707 = 0.293     │
  │ 典型情况 p = 0.1, γ = 0 (或反之):                            │
  │   f_FB = 0.1 − 0.1 = 0                                     │
  ├─────────────────────────────────────────────────────────────┤
  │ f_FB 的 RMS 值 ≈ 0.03                                       │
  │ MSE ≈ 0.03² ≈ 1×10⁻³                                       │
  └─────────────────────────────────────────────────────────────┘
```

**L_FB 小的根本原因**：
1. 输入输出范围受限（p, γ ∈ [0, 1]），FB 函数的输出也在 [0, 0.3] 以内
2. `tanh²` 天然保证非负性，满足了一半的条件
3. 物理上 p 和 γ 天然倾向于在空间上分离，只需要网络学习"哪里应该空化"
4. FB 函数在 p ≫ γ 或 γ ≫ p 时非常接近 0，对小的不完美不敏感

### 7.4 梯度竞争：为什么简单的 sum(loss) 有问题

在新代码中（已移除 PCGrad 和自适应权重），总损失是直接求和：

```
L_total = L_Reynolds + L_BC + L_FB
        ≈ 1×10⁶   + 1×10⁻⁴ + 1×10⁻³
```

|∇L_Reynolds| ≈ 10³ ~ 10⁵（来自二阶 autograd 的大梯度）
|∇L_FB|       ≈ 10⁻² ~ 10⁰（来自输出层的小梯度）
|∇L_BC|       ≈ 10⁻³ ~ 10⁻¹

**L_Reynolds 的梯度完全淹没了 L_FB 和 L_BC 的梯度**。这意味着优化器几乎只看到雷诺方程的约束，JFO 条件和 BC 的信号被淹没。

这在训练早期可能是好事（先满足物理），但在后期可能导致：
- L_FB 虽然数值小，但**梯度方向可能和 L_Reynolds 冲突**
- 在缺失 PCGrad 的情况下，小的 L_FB 梯度被完全忽略
- 模型可能学习到一个"物理上满足雷诺方程但 JFO 条件略有违背"的解

---

## 8. 核心难点总结

### 难点 1：多尺度 PDE 残差（数量级鸿沟）

```
  Λ·∂H/∂θ ≈ 86,400
  H³·∂²p/∂R² ≈ 3~200
  ———————————————————
  差距 ~ 430~28,800 倍
```

网络必须学习产生极精细的压力分布来抵消巨大的楔形效应源项。这是一个**高对流主导的对流-扩散问题**（Λ = 604），对网络的表达能力和光滑性要求极高。

### 难点 2：H³ 系数对比度（64×）

槽区 H³=64，台地区 H³=1。同一方程在不同区域的系数差异达 64 倍，导致：
- 槽区的残差主导了总损失
- 台地区的残差信号被淹没
- 配点分布（RAD）虽然部分缓解了这个问题，但不能根治

### 难点 3：自由空化边界

空化区的位置和形状是**未知的先验**。JFO 条件是一个互补约束，它在全膜区和空化区之间定义了一条**自由边界**。这条边界的位置由物理决定，但网络必须同时学习压力场和空化场的空间分布——且两者在边界处不光滑（一个降为 0，一个升起来）。

### 难点 4：Loss 之间的量级不匹配

```
L_Reynolds : L_FB : L_BC ≈ 10⁶ : 10⁻³ : 10⁻⁴
```

梯度幅值差距 5~9 个数量级。简单求和会导致小损失项的梯度被淹没。这也是为什么原 TF 版本引入了 PCGrad + 自适应权重的组合策略——PCGrad 消解方向冲突，自适应权重平衡量级差异。

### 难点 5：二阶 autograd 的数值挑战

雷诺方程需要 `∂²p/∂R²` 和 `∂²p/∂θ²`——这意味着自动求导要穿透网络两次。网络的平滑性、激活函数的选择（tanh）、权重的初始化都对二阶导数的质量有决定性影响。训练初期的随机权重产生的高频噪声在二阶导数中被放大。
