## 2. 训练阶段输出

> 脚本：`reynold_pinn.py`
> 输出位置：`output/figures/reynolds_pinn_N4900_iter80000/`

训练完成后在 `output/figures/` 目录下自动生成 4 张图，全部基于 **PINN 在 401×401 高分辨率网格上的预测**。

### 2.1 压力分布云图

**文件名**：`*_pressure_contour.png`

**图像内容**：

- X 轴：无量纲径向坐标 `R`（范围约 0.904 ~ 1.0，对应物理半径 47~52 mm）
- Y 轴：无量纲角度坐标 `θ`（范围 0 ~ 1.047 rad = 60°，即 **1/6 个圆周**的一个周期扇形）
- 颜色/云图值：**无量纲压力 `P`**

**配色方案**：`RdYlBu_r`（红-黄-蓝倒序），即：
| 颜色 | 含义 |
|------|------|
| 🔴 红色/橙红色 | 高压区（P 较大） |
| 🟡 黄色/浅色 | 中等压力 |
| 🔵 蓝色/深蓝 | 低压区（P 接近 0） |

**如何看图**：

1. **螺旋带状高压区**：应该能看到从外径（R≈1.0）向内的螺旋形高压带——这是螺旋槽的**泵吸效应**将油从外径泵入并压缩的结果
2. **压力最大值位置**：高压峰值应出现在螺旋槽末端的台面区（land region），即槽的"收敛楔"位置
3. **低压空化区**：深蓝色区域（P≈0）对应空化区——螺旋槽的发散区（槽突然结束处），油膜膨胀导致压力降到蒸气压力
4. **内外边界**：R 最小值边界（左边缘）= P_i ≈ 0.0667，R 最大值边界（右边缘）= P_o ≈ 0.1000（对应内外径的供油/出油压力）
5. **好的结果特征**：
   - 高压带和低压带呈清晰的螺旋条纹，周期性与 K=6 的对称性一致
   - 从外径到内径的压力过渡平滑，无锯齿或噪点
   - 颜色过渡连续，无突变（除了槽边界附近）

### 2.2 空化率分布云图

**文件名**：`*_cavitation_contour.png`

**图像内容**：

- X 轴：无量纲径向坐标 `R`
- Y 轴：无量纲角度坐标 `θ`
- 颜色/云图值：**空化率 `γ`**（范围 [0, ~1]）

**配色方案**：同样 `RdYlBu_r`：
| 颜色 | 含义 |
|------|------|
| 🔵 蓝色 | γ≈0（全膜区，无空化） |
| 🔴 红色 | γ>0（空化区，存在气泡） |

**如何看图**：

1. **空化区的空间位置**：红色/暖色区域应出现在螺旋槽发散的**下风侧**（downstream side of groove）——即槽的出口边界附近，这里膜厚突然增大，油膜"来不及填充"
2. **与压力图的互补关系**：这张图应该和压力图呈**镜像关系**——压力高的地方空化率为零（红色区域对应压力图的蓝色/深蓝区）
3. **空化条带**：理想情况下应看到清晰的螺旋形空化条带，每条宽度约占槽宽比 0.5 对应的一部分
4. **好的结果特征**：
   - 空化区域（γ>0）与高压区（P>0）没有重叠——这是 JFO 互补条件 `P·γ=0` 的直观体现
   - γ 值在空化区应接近 1（完全空化），在全膜区为 0
   - 空化边界清晰锐利，过渡带窄

### 2.3 膜厚分布图

**文件名**：`*_H_only.png`

**图像内容**：

- X 轴：无量纲径向坐标 `R`
- Y 轴：无量纲角度坐标 `θ`
- 颜色/云图值：**无量纲膜厚 `H`**

**配色方案**：`RdYlBu_r`：
| 颜色 | 含义 |
|------|------|
| 🔴 红色 | 深槽区（H = 1 + h_texture/h_i = 4.0） |
| 🔵 蓝色 | 台面区（H = 1.0，平衡膜厚） |

**如何看图**：

1. **螺旋槽的直观可视化**：这张图直接展示了**螺旋槽的几何形状**——红色的螺旋带就是刻在下盘表面的槽
2. **槽的形态**：
   - 呈对数螺线形状（θ_sym(R) = ln(R/r_g)/tan(α) + θ_offset）
   - 从内径 R_d_1 处开始，向外径 R_d_2 处延伸（本项目 R_d_2 已超出外径，即槽完全贯通到外边界）
   - 每条槽占 60° 扇形周期的一半宽度（groove_ratio=0.5）
   - 共 6 条槽（对应 K=6 个周期），但由于只计算 1/6 扇形（一个周期），图中通常只显示 1 条完整槽
3. **过渡带（关键设计点）**：槽边界从红到蓝的过渡不是突变的，而是有**渐变带**。这来自 sigmoid 平滑函数的过渡宽度 ξ_R 和 ξ_θ
4. **好的结果特征**：
   - 槽的几何形状与设计参数（螺旋角 3°、起始/结束位置、宽度比 0.5）完全吻合
   - 过渡带宽度适中（不太宽也不太窄）——过宽则物理精度低，过窄则导致 PDE 残差在边界爆炸

### 2.4 训练损失曲线

**文件名**：`*_loss_log.png`

**图像内容**：

- X 轴：训练 epoch 数
- Y 轴：**总损失值（对数坐标 log scale）**
- 蓝色实线：Total Loss 随训练的变化轨迹

**如何看图**：

1. **整体趋势**：损失应呈**单调下降**趋势（注意是对数坐标，视觉上的"直线下降"实际是指数级衰减）
2. **阶段过渡**：4 个 LR 阶段（各 20000 步）的转换处可能看到损失曲线的"拐点"——每次 LR 骤降时损失往往先略有上升再快速下降
3. **RAD 细化信号**：每个阶段内的 4 轮 RAD 自适应配点细化（每 5000 步）可能表现为损失曲线的周期性"尖刺"——因为配点分布改变后，损失函数的目标变了（更多点集中在难区），损失可能短暂上升
4. **好的结果特征**：
   - 损失最终降到 10⁻⁴ 量级或更低
   - 没有长期平台（plateau）——说明学习率调度有效
   - 曲线末段趋平但仍在缓慢下降——说明训练已接近收敛但没有完全停滞
   - 无发散跳变（loss 突然增大几个数量级）——如有，说明数值不稳定

## 3. 对比测评阶段输出

> 脚本：`compare_fem_pinn_final.py`
> 输出位置：`output/comparison_results/`

测评阶段将**PINN 预测**与**FEM 参考解**（201×201 网格，`p_FBNS.txt` / `g_FBNS.txt`）在同一网格点上进行系统对比。FEM 是工程界的"金标准"，本项目将其作为 **ground truth**。

---

### 3.1 FEM与PINN场分布对比

**文件名**：`fig1a_P_fem.png`, `fig1b_P_pinn.png`, `fig1c_G_fem.png`, `fig1d_G_pinn.png`

**图像内容**：4 张并排云图，将 FEM 和 PINN 的 P 场和 g 场**直观放在一起对比**。

| 图 | 内容 | 期望看到 |
|----|------|---------|
| `fig1a` | FEM 压力场 P(R,θ) | 参考标准——"Ground Truth" |
| `fig1b` | PINN 压力场 P̂(R,θ) | 应与 `fig1a` **高度相似** |
| `fig1c` | FEM 空化率场 g(R,θ) | 参考标准 |
| `fig1d` | PINN 空化率场 ĝ(R,θ) | 应与 `fig1c` **尽可能相似** |

**配色方案**：统一使用 `RdYlBu_r`（红=高值，蓝=低值）

**如何看图——三步快速评估法**：

1. **第一眼扫视**：`fig1a` vs `fig1b` 的颜色分布是否"像"？螺旋条纹的方向、位置、粗细是否一致？颜色深浅（值的大小）是否匹配？
2. **细节对比**：沿着螺旋槽走向，高压带（红色）和低压带（蓝色）的空间位置在 FEM 和 PINN 之间是否对齐？特别是螺旋槽出口附近的低压（空化）区
3. **边缘检查**：内边界（R 最小处）和外边界（R 最大处）的颜色是否一致？边界条件是精确满足的硬约束，FEM 和 PINN 在边界处应该完全相同

**好的结果特征**：
- `fig1a` 和 `fig1b`（压力）几乎难以区分——说明压力场精度极高（本项目实际达到 ~1.4% L2 误差）
- `fig1c` 和 `fig1d`（空化率）大体相似但可能存在细节差异——空化边界对 PINN 是固有难点
- 压力图的螺旋形高压带在两张图中长度、宽度、位置一致
- 空化图的蓝色（全膜）和红色（空化）区域的空间覆盖大体一致

### 3.2 逐点误差热力图

**文件名**：`fig2a_err_P.png`, `fig2b_err_G.png`

**图像内容**：
- `fig2a`：**压力逐点绝对误差** |P_FEM − P̂_PINN|，使用 `RdYlBu_r` 配色
- `fig2b`：**空化率逐点绝对误差** |g_FEM − ĝ_PINN|（仅在 FEM 空化区 g>0.01 处显示），使用 `hot_r` 配色（白→黄→红→黑）

**配色方案**：

| 图 | 配色 | 颜色含义 |
|----|------|---------|
| fig2a（压力误差） | `RdYlBu_r` | 🔵蓝=误差小（好），🔴红=误差大（差） |
| fig2b（空化误差） | `hot_r` | ⬜白/黄=误差小，🟥红/黑=误差大 |

**如何看图——这是最重要的诊断图**：

1. **误差的空间分布**：误差在哪里大？理想情况下应该均匀地小（全图偏蓝/白），但实际上：
   - **槽边界附近**误差往往较大——因为膜厚 H 在槽边界处变化剧烈，PDE 残差也大
   - **空化边界附近**误差往往最大——这是自由边界（内边界），是 PINN 最困难的区域（`fig2b` 的红色/黑色区域如集中在空化-全膜过渡带，说明边界定位不准）
   - **全膜区内部**误差应很小（~10⁻⁴ 量级）
2. **误差的数值量级**：看 colorbar 的刻度
   - 压力误差 `fig2a` 最大约 0.01~0.03（对应 ~7.2% L∞ 误差），大部分区域 << 0.01（好）
   - 空化率误差 `fig2b` 最大可达 0.6（空化边界附近），超出空化边界则误差回到低值
3. **系统性偏差检测**：如果某个区域（如外径附近、某个角度范围）持续偏红，说明 PINN 在该区域存在**系统性偏差**（而非随机误差），需要针对性改进

**好的结果特征**：
- `fig2a` 整体偏蓝，仅螺旋槽边界处有浅色过渡带
- `fig2b` 的"热点"（高误差）仅沿空化边界呈细线状分布，而非大片红色块
- 两个误差图都不存在孤立的"红色斑块"（那可能是 PINN 的局部收敛失败）

### 3.3 空化边界对比

**文件名**：`fig3_cavitation_boundary.png`

**图像内容**：
- **背景**：PINN 预测的空化率 ĝ 作为半透明蓝色云图（`Blues` colormap, alpha=0.5）
- **红色实线**：FEM 的空化边界（g = 10⁻⁶ 等值线，即 g 从 0 变为正值的临界线）
- **蓝色虚线**：PINN 的空化边界（同样是 ĝ = 10⁻⁶ 等值线）

**配色/线型约定**：
| 元素 | 颜色 | 线型 | 含义 |
|------|------|------|------|
| 背景云图 | 蓝色调半透明 | — | PINN 空化率分布 |
| 红色实线 | 🔴 红 | `—` 实线 | **FEM 空化边界**（参考标准） |
| 蓝色虚线 | 🔵 蓝 | `--` 虚线 | **PINN 空化边界**（待评估） |

**如何看图——空化边界是核心难点**：

1. **两条线的整体走势是否一致**：沿螺旋槽方向，红线和蓝线的形状和位置是否重叠？
2. **偏差的位置特征**：
   - PINN 边界偏内（蓝线在红线左侧/内侧）→ PINN 预测的空化区偏小
   - PINN 边界偏外（蓝线在红线右侧/外侧）→ PINN 预测的空化区偏大
   - 哪种偏差更大，在哪个 R 范围偏差最大？
3. **边界闭合情况**：空化区应该是一个封闭的区域（气泡被全膜油包围），两条线是否都能形成闭合轮廓？
4. **背景蓝色深浅**：半透明蓝色云图的深浅指示了空化率的大小，观察 PINN 空化率在越过边界后是否能快速上升到接近 1

**好的结果特征**：
- 红线和蓝线基本重叠，最大偏差在毫米级（对应 θ 偏差 < 0.05 rad）
- 两条线都是光滑曲线（无锯齿）
- 空化区域的形状（闭合轮廓）在 FEM 和 PINN 之间一致

### 3.4 压力等值线叠加

**文件名**：`fig4_pressure_contour_overlay.png`

**图像内容**：
- **红色实线**：FEM 的压力等值线（15 条等间距等值线，从 P_min 到 P_max）
- **蓝色虚线**：PINN 的压力等值线（相同 15 个等值线水平）
- 每条等值线标注了其 P 值

**配色/线型约定**：
| 元素 | 颜色 | 线型 | 含义 |
|------|------|------|------|
| 红色实线 + 标注 | 🔴 红 | `—` 实线 | **FEM 压力等值线**（参考标准） |
| 蓝色虚线 | 🔵 蓝 | `--` 虚线 | **PINN 压力等值线**（待评估） |

**如何看图**：

1. **等值线重叠度**：同一 P 值下，红线和蓝线是否重叠？这是压力场全局精度的直观指标
2. **偏差最大的区域**：哪些地方的蓝线和红线分离开？
   - 高压峰值附近（等值线密集区）→ 峰值预测偏差
   - 空化边界附近（低压等值线，P≈0）→ 空化-全膜过渡区精度
3. **等值线间距**：等值线密集处 = 压力梯度大，稀疏处 = 压力平缓
4. **形状一致性**：等值线的走向和弯曲形状是否一致？即使位置有微小偏移，形状一致说明 PINN 学到了正确的物理规律

**好的结果特征**：
- 大部分等值线红蓝重叠，肉眼难以区分
- 仅在低压区（P≈0 的等值线）附近有轻微偏差——这是合理的（空化边界处数值不稳定）

|------|----------|------|

### 3.8 JFO互补条件

**文件名**：`fig8_jfo_complement.png`

**图像内容**：2 张 P-vs-g 散点图，用于检验物理约束 `P·g = 0`（压力和空化率不能同时为正）。

| 子图 | 内容 |
|------|------|
| 左图 | FEM：P vs g |
| 右图 | PINN：P vs g |

**配色约定**：
| 元素 | 含义 |
|------|------|
| 散点颜色（hot_r colormap） | 每个点的 `P·g` 乘积值（越红=乘积越大=约束违反越严重） |
| 灰色虚线 | P=0 和 g=0 的参考线 |
| max(P·g) 标注 | 最大约束违反量（理想值=0） |

**如何看图——物理约束遵守程度**：

1. **理想的 JFO 互补**：所有点应沿**两条轴**分布：
   - 沿水平轴（P>0, g=0）→ 全膜区的点
   - 沿垂直轴（P≈0, g>0）→ 空化区的点
   - **不应有任何点同时远离两条轴**（即不应有 P>0 且 g>0 的点）
2. **FEM 作为参照**：FEM 的散点图（左图）应在两轴上完美分布，展示了"理想状态"
3. **PINN 的违反**：PINN 散点图（右图）中，如果有明显的"内部"散布（不在两轴上），说明 PINN 在空化边界附近违反了互补条件
4. **max(P·g) 数值**：越小越好，本项目实际值 ~3.2×10⁻³，量级很小，说明约束基本满足

**好的结果特征**：
- 散点图呈清晰的"L"形分布（沿两轴）
- max(P·g) < 10⁻²（很小）
- 非轴上的散点极少，颜色浅（说明乘积小）
- PINN 的 L 形与 FEM 的 L 形相似

#### 组 4：分区域空化率 g 误差
| 指标 | 含义 | 本项目实际值 | 解读 |
